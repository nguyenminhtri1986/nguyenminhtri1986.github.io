1. GIỚI THIỆU

 Để thuận tiện học thiết kế web, chúng ta cần cài đặt máy tính của mình một chương trình web server để máy tính của mình trở thành một localhost nhằm upload các website và xem các trang web.

 Các gói cài đặt có thể là:

-       AMPP: Apache Server - MySQL - PHP - phpMyAdmin

-       WAMPP: Windows - Apache Server - MySQL - PHP - phpMyAdmin

-       LAMPP: Linux - Apache Server - MySQL - PHP - phpMyAdmin

-       MAMPP: Macintosh - Apache Server - MySQL - PHP - phpMyAdmin

-       XAMPP: HĐH - Apache Server - MySQL - PHP - phpMyAdmin

   VIII.            LINK

 Khi ta nhấp chuột lên một chuỗi văn bản hoặc một hình ảnh có liên kết thì sẽ chuyển đến:

·            Một file tài liệu, hoặc một file khác như word, excel, PDF, phim, nhạc, ZIP, RAR

·            Một trang web khác trong cùng một Website (gọi là Internal Link).

·            Một trang web của một website khác (gọi là External Link).

·            Một vị trí nào đó trong trang web hiện hành (gọi là Link to Anchore name).

·            Trên ảnh còn có nhiều vị trí để link đến các trang  khác nhau. Các vị trí này gọi là Hot spot link.

·            Email link:  Khi nhấp vào một dòng địa chỉ email thì Gởi Email.

·            Null link: không đi đâu cả.

1- Tạo Link

Hình 99

·            Chọn hình ảnh hoặc chuỗi văn bản mà muốn tạo siêu liên kết.

·            Trên thẻ Common, click nút Hyperlink

·            Trong hộp Hyperlink ta nhập tên đường dẫn liên kết, cửa sổ target, title hiển thị, access key (phím tắt), tab index (số chỉ mục của phím Tab).

2 - Tạo Internal Link

Hình 100

·            Chọn hình ảnh hoặc chuỗi văn bản mà muốn tạo siêu liên kết.

·            Nhập trực tiếp tên file vào hộp link

·            Hoặc click nút Browse  For File rồi chọn trang web trong site đang thiết kế.

·            Hoặc Kéo rê nút Point to File đến tên file nằm trong panel Files.

·            Tùy chọn Target= _blank khi muốn trang web hiện ra trong cửa sổ mới.

·            Tùy chọn Target= _self  khi muốn trang web hiện ra trong cửa sổ hiện hành.

·            Tùy chọn Target= _parent  khi muốn trang web hiện ở trang cha.

·            Tùy chọn Target= _top khi muốn trang web hiện ra ở toàn trang.

3 - Tạo External Link

Hình 101

·            Chọn hình ảnh hoặc chuỗi văn bản mà muốn tạo siêu liên kết.
